# Next Obsidian plugin release

## Added

- Added a settings warning when files are blocked because they exceed the sync size limit.

## Changed

- Reduced the minimum remote vault password length to 12 characters.
- Improved the deleted files restore modal with a select-all control and in-list pagination.
- Improved local sync reconciliation so large vault scans batch metadata updates more efficiently while preserving pending file changes.
- Improved the settings sync and storage status display so it updates in place while sync is running.
